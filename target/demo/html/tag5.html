시멘틱 태그?

HTML5에서 웹문서에 의미를 부여하는 태그를 의미함.(div) 시멘틱 태그 사용 전에는 div 태그에 id 혹은 클래스 통해서 성격을 지정. 단순히 보기에는 어떤 부분을 담당하는 div 태그인지 구분이 잘 안됨. 그런 이유로 웹문서 의미를 부여하는 시멘틱 태그를 사용하기 시작됐음.

주요 시멘틱 태그(layout 측면에서)

  • header: 웹 문서에 머릿말(사이트에 대한 소개, 제목, 로고)
  • nav: 네비게이션(메뉴)
  • section: 웹 문서를 장(Chapter)으로 볼 때 절을 구분하는 태그
  • aside: 문서의 보조 내용(광고, 즐겨찾기, 링크(네비게이션))
  • footer: 문서의 꼬릿말(작성자, 저작권, 개인정보 보호)
